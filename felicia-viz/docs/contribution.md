## Development tips

### Webpack Dev Server

```bash
cd backend
npm run start:dev
```

## Coding style

We use [prettier](https://prettier.io/) as a formatting tool, we provide the command to format codes like below.

```bash
npm run format
```

## Lint

We use [eslint](https://eslint.org/) as a lint tool, we provide the comamnd to lint like belows.

```bash
npm run lint
```
