# How to use

## Command Panel

<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>p</kbd> will opens command panel!

## Add View

1. Click **+** button.
2. Add view thorugh command panel.

## Move view

If view is focused, you can move view with arrow keys, or drag.

## Remove View

<kbd>Delete</kbd> will remove the focused view.
